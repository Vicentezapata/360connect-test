Historias de Usuario

Phase 0 — Infra & Migración

4 sem

Stack base levantado en Docker Compose. Migración de fotos legadas a MinIO. CI/CD operativo.

10 tareas ~17d esfuerzo Dev1 + Dev2
Entregables: Docker Compose · Nginx SSL · MinIO buckets · Prisma schema · install.sh · migrate-photos.js · CI/CD

Phase 1 — API Core

8 sem

Todos los endpoints NestJS. Auth JWT + rate limiting. RBAC con scope geográfico. API reportes de campo.

18 tareas ~34d esfuerzo Dev1
Entregables: Auth JWT · RBAC geo · CRUD branches/spaces · Upload MinIO · BullMQ · API hotspots polimórficos · API reportes · Presigned URLs

Phase 2 — Viewer Público

9 sem

Tour 360° con menú 9 ítems, paneles de equipos, hotspots pin y polígono, PWA offline completo.

14 tareas ~31d esfuerzo Dev2
Entregables: PSV viewer · Menú 9 ítems · Hotspot poligonal · Mini-mapa · V-08 reporte campo · Background Sync API · Offline PWA

Phase 3 — Admin Portal

15 sem

Portal admin completo: galería, editor hotspots pin+polígono, planimetría canvas, biblioteca, inspecciones, usuarios RBAC.

17 tareas ~47d esfuerzo Dev1 + Dev2
Entregables: Dashboard · Editor hotspots · Planimetría Konva · Documentos polimórficos · A-17 inspecciones · RBAC scope geo · Offline admin

Phase 4 — Módulo Campo

6 sem

Ciclo completo de inspección offline para técnicos. Registro de fallas con fotos. Background Sync API.

6 tareas ~13d esfuerzo Dev1 + Dev2
Entregables: API POST /reports/field · V-08 formulario + cámara · outbox_reportes IndexedDB · Background Sync · Workflow A-17

Phase 5 — Operaciones

3 sem

Hardening OWASP, backup automático, logs estructurados, runbook de operaciones.

5 tareas ~8d esfuerzo Dev1
Entregables: pg_dump → MinIO · Security headers · OWASP hardening · Winston logs · Runbook .env

Phase 6 — QA & Launch

8 sem

E2E Playwright (viewer + admin + offline + reportes), k6 load test, Lighthouse, deploy producción.

11 tareas ~26d esfuerzo QA + Dev1
Entregables: 6 suites E2E Playwright · k6 (50 usuarios concurrentes) · Lighthouse ≥ 90 · ZAP audit · Deploy prod

API Lista

Todos los endpoints NestJS documentados y testeados. Backend completo.

Viewer Beta

Admin Beta
